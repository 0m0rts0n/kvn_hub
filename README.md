# kvn_hub

Минимальный репозиторий для GitHub Pages, где страница читает содержимое файла `data.txt`.

## Структура

- `index.html` - страница GitHub Pages;
- `script.js` - загрузка `data.txt` через `fetch()`;
- `styles.css` - оформление страницы;
- `data.txt` - текстовые данные, которые выводятся на странице.

## Публикация на GitHub Pages

1. Создайте репозиторий `kvn_hub`.
2. Загрузите содержимое этой папки в ветку `main`.
3. В настройках репозитория откройте `Settings -> Pages`.
4. В `Build and deployment` выберите `Deploy from a branch`.
5. Укажите ветку `main` и папку `/ (root)`.

После публикации страница будет доступна по адресу вида:

`https://0m0rts0n.github.io/kvn_hub/`

Важно:

- для личного аккаунта GitHub публикация Pages из приватного репозитория доступна только на `GitHub Pro` и выше;
- на `GitHub Free` для GitHub Pages репозиторий должен быть публичным;
- по документации GitHub приватная публикация Pages поддерживается только для организаций на `GitHub Enterprise Cloud`.
